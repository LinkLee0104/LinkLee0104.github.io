写在Spring 2020结束之际

发表于 2020-05-23

一个不平凡的学期，在平凡中匆匆收尾。尽管怠于执笔，这段令人印象深刻的时光也该写点东西总结。

学习

算法（Intro to Algorithms）和软件原理（Principles of Software）两门专业课，一门理论一门实践刚好形成互补。最小生成树、动态规划、NP问题、多态和设计类型在未来学习和工作也有相当多的应用场合；线性代数（Linear Algebra）则是纯粹作为未来上机器学习的prerequisite，课程内容以及教授的板书都相当抽象，所幸靠看书+请教大神苟了过来；声乐基础（Fundamentals of Music & Sound）没想象的那么理论，但是各种奇奇怪怪的作业实在占据了自己不少时间。尽管简单易懂，这种耗时耗力，还要你拍抖音的奇葩“水课”还是放到以后没那么忙时再上会更好。
身边更多的是拿到教授推荐信、绩点拉满的学神，自己并没有什么值得骄傲的资本。未来还有更高阶的课程去挑战，现在也只是刚刚为它们打好基础。

研究

二月初学校的数据研究部门开放了一项整理国内抗击疫情物资的研究，由于面试时表现平平遗憾落选。期末结束后这个部门又放出了好几项暑假期间的research，其中有一项便是关于美国新冠疫情下学校复工的预计。给项目负责人打了一通Webex后了解到这是一个侧重数据可视化的面板（Dashboard），其中后端统计用的是R语言，即便没有先前经验上手也不成问题。面试无异于自我介绍和拉拉家常，不过这次自己格外注重谈吐，和负责人聊了好些简历上做项目的经验。几天后收到邮件确认录取，成功如愿以偿。
RPI本科生研究被单独算作一个叫URP的title，在CS专业课程中可以当Capstone。这样的话未来选课就不必满足特定的方向，没有诸如理论算法、系统软件等专门的课程方向限制。因此这次研究最后算了学分记在暑假学期上，充实阅历的同时还能拉一波绩点。
这样算下来暑假就有20分的课业量了，美汁儿汁儿。

生活

居家生活两个月后才发现这种生活格外枯燥无聊，就真是那种养猪般的生活。好在可以和两位室友日常谈笑风生+互相鬼畜，才能愉快度过这漫长的时光，不至于自闭的同时也增加了不少奇怪的知识。
四月中旬天天刷BestBuy，跟风抢到了一台Switch，因此也开始入坑动物森友会和塞尔达传说。果然还是这种开放世界+非线性探索游戏最令人满意，比起自力更生的Minecraft多了剧情挑战，刷玩后还能当当老流氓到处炸鱼。这也应该是自己的第一次主机体验，N社游戏的各种体感操作玩得实在不亦乐乎。
任天堂就TM是世界的主宰！

回国

由于CAAC的“五个一”政策，现在飞往中国的航班屈指可数，然后学校也没有确切通知秋季学习的安排。现在也只能脑补一下仅剩的航线怎么飞了。
直航买不起也买不到，首先排除；其次便是第三国转机，NRT或ICN应该能走，LHR或申根区貌似有美签转机也不是问题。联程现在基本没戏，美国大城市->转机点->中国目的地两张机票只能分开买。要是能订到统一航空联盟/有行李直挂协议的航司那最好不过，不然要入境取行李的话就真的凉凉。反正回去怎么都得隔离十四天，能不能到NKG根本不必考虑。只要别摊上扔行李那种事就好。
时运不济，自求多福吧！

git涉及branch的一些用法

发表于 2019-10-25

由于做RCOS项目时进行了一次JS->Python->JS的技术变动，现在项目组的后端github repo里有了好几个branch，包括新版的JS后端和之前的Python版。因此现在也就要重新把代码从另外一个branch里pull下来。
git可以做到从repo里同步最新的代码到本地，一句git clone就可以：
git clone <git-repo>
这里如果没有指定branch的话，clone下来直接是origin。比如我们现在要把另外一个分支pull下来，在git clone后面就要指定你的branch：
git clone -b <branch-name> <git-repo>
现在你就会发现你的目录里多了个文件夹，里面就是新鲜出炉的代码了。然后要是有什么更新的话，进入目录一句git pull就能同步。
不过这样还是从别的branch新clone了一下，现在我们要做的是把新branch的内容同步到原来的工作目录里。所以我们进入原有的工作目录，然后
git remote -a
现在就能看到我们目前所在的branch以及remote repo里所有的branch。比如我们要切换到es7这个branch，现在就能从指定的新branch用checkout把代码同步下来了：
git checkout origin/es7
稍等一会儿后你的目录里就是新branch里的内容了。

第41届NSBE/SHPE校园招聘会参加体验

发表于 2019-09-23

人生第一次参加career fair，着实该写点东西纪念下。
大概是去年3月的时候，RPI有过一次小型招聘会。然而彼时因为对于校招没有任何清晰的认识，没准备简历就跑去玩，结果自然毫无所获。九月开学后，想着今年career fair就在20号，便开始准备自己的首次校招了。
首先便是得有张像样的简历。得益于这学期上FoCS用到的LaTeX，简历这种一面纸简而概之的文件可以更好地利用页面。学校官网甚至都有提供几个90年代的LaTeX模板方便学生排版，因此页面布局分分钟搞定。内容无异于自卖自夸一番，而且因为首次写成需要大规模修改。与其随大流地去CCPD改简历，我去了CS系的荣誉社团UPE。相比于CCPD根据RPI大多工科学生的情况给出大概的修改意见，UPE会根据CS的就业方向更针对地进行修改。最后打印了20份装进文件袋，简历便准备好了。
招聘会对于着装方面也有一定要求，一般都是business casual。因为身上就只有一套黑色西装，所以也就formal一下穿正装去校招了。实战下来的确相当得体，不过西装革履在人头攒动的会场里真的令人窒息。

别再摸鱼了

发表于 2019-08-07

超级划水的一个暑假，水到连自己的博客都懒于维护。
被数据结构折磨的一个学期终于结束。因为听信了一些令人恐慌的传闻，外加自己欠缺学习和时间管理的技巧，成绩最终不尽人意。回国不能免俗地到处见老同学吃饭逛街。六月在北京上了一个月夏校，幸会了同年级大神@NeverBehave，夏校结束后他还来南京逗留了一两天。后来又是被拉到杭州玩了几天，回家后没过几周暑假便匆匆结束。四个月说来长，实则在摸鱼中被挥霍。除了象征性地刷了几题leetcode以外，什么有意思的事情也没做。
博客于此文之前一直在本地酝酿没有公开。原本准备部署在自己开的Vultr，却因为$0x40=30导致Vultr近乎所有的IP在国内访问受限。兴许可以效仿同学搞home-hosting，可是购买设备成本不低，学期结束后的搬家也会成一场噩梦。还是干脆尽快把博客部署到github算了。
期待下学期旁边有快餐店和商店，有厨房的新宿舍。

你没有玩过的全新博客

发表于 2019-02-18

我就是知识贫乏而死，死在作业堆里，也不会浪费时间开博客当作家！
嗯~真香！

契机

面向Google编程的时候，经常能查到一些博主的网站，由此发现这一“网上冲浪”时期的产物仍然存活于朋友圈和空间之外，是一片难得可以尽情表达的净土。
鉴于第三方平台局限较多，就准备自己弄一个GitHub Repo/服务器开博。这样方便找到了更好的一套博客系统后迁移，也可以对自己的内容有完全的掌控。

现实

很多事情和高中时想象的大学生活差别太大了。明明课程没那么紧张，作业量和压力却远远高于以前。当然最恐怖的还是peer pressure：人家周末两天写完了你一周才肝完的大作业，开心地去打游戏/做项目了，最后还早早进了校开发组，GitHub上全是star，留你在这里干瞪眼。缓过神来，你的程序还全是bug呢！

没事就写写当下，留给未来的自己细细品味吧。
